📱 Solutions pour Android

1. Ajouter au Home Screen (Solution Simple)

Depuis Chrome/Firefox/Edge :

  1. Ouvre ton fichier HTML dans le navigateur

  2. Menu ⋮ (3 points) → "Ajouter à l'écran d'accueil"

  3. Un raccourci s'ajoute comme une vraie app

⚠️ Problème : Ça crée juste un favori, les données en mémoire se perdent quand tu fermes l'app.

2. Transformer en PWA (Progressive Web App) - SOLUTION IDÉALE ✅

Pour que ton app garde ses données et fonctionne même hors ligne, il faut ajouter 2 petits fichiers à ton HTML :

Fichier 1 : manifest.json

json
{
  "name": "Mon Application",
  "short_name": "MonApp",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#000000",
  "icons": [
    {
      "src": "icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    }
  ]
}

Fichier 2 : serviceWorker.js

javascript
self.addEventListener("fetch", function(event) {
  console.log('Service Worker actif');
});

Modifier ton HTML (ajoute dans <head>) :

xml
<link rel="manifest" href="manifest.json">
<script>
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('./serviceWorker.js')
    .then(reg => console.log('Service Worker enregistré'))
    .catch(err => console.log('Erreur:', err));
}
</script>

3. Solution ALTERNATIVE : Stockage avec IndexedDB

Si tu veux éviter les PWA complexes, remplace le stockage en mémoire par IndexedDB (fonctionne en sandbox contrairement à localStorage).
